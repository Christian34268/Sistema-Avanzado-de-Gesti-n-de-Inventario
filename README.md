# Sistema-de-Gesti-n-de-Inventarios
# 📦 Sistema Avanzado de Gestión de Inventario

Sistema desarrollado en Python para la asignatura de Programación Orientada a Objetos.

## 🔄 Mejoras 
Esta versión optimiza el manejo de datos utilizando **diccionarios** para almacenar productos (búsqueda eficiente por ID) y **conjuntos** para validar IDs duplicados instantáneamente. Se emplean **tuplas** para serializar datos en archivos. Se ha mejorado la experiencia con emojis en consola y secciones comentadas.

El sistema carga automáticamente el inventario al iniciar y guarda los cambios tras cada operación en `inventario.txt`, manejando errores de archivo de forma segura.
